# 原神 育成コンパス

攻略サイトやYouTube解説動画など、インターネット上で一般に公開されている『原神』の育成情報をもとに、人気・主要キャラクター25体の理想ステータス・おすすめ聖遺物・おすすめ武器・深境螺旋(スパイラルアビス)向けおすすめ編成をまとめた非公式ファンサイトです。

## 公開ページ

GitHub Pages で公開しています(有効化後、下記URLでアクセスできます)。

## 内容

- キャラクターごとの理想ステータス・おすすめ聖遺物・おすすめ武器
- 属性・武器種でのフィルタ、名前検索
- 深境螺旋向けおすすめ編成(2パターン)
- 実装バージョンが新しい順の並び替え

## 免責事項

本サイトは、ゲーム攻略サイトやYouTube解説動画など一般に公開されている情報をもとに編集部にて整理・要約したものです。数値やおすすめ装備は各情報が公開された時点のバージョンに基づくため、実際の最新環境とは異なる場合があります。当サイトはゲーム公式運営および参照先の各サイト・チャンネルとは関係のない非公式のファンサイトです。
